
# 🚀 AI Study Planner

AI-based system that generates optimized study schedules based on subject difficulty and available time.

## ❗ Problem Statement
Students often struggle to allocate study time effectively, leading to inefficient learning.

## ✨ Features
- 📊 Priority-based scheduling
- ⚡ Dynamic time allocation
- 🧠 Optimization logic

## 🧠 How It Works
Subjects are weighted based on difficulty, and total study time is distributed proportionally.

## 🌍 Why This Matters
Efficient planning improves learning outcomes and reduces stress.

## 🧠 Machine Learning Integration

This project incorporates a Linear Regression model to predict study time allocation based on subject difficulty.

This demonstrates a transition from rule-based planning to data-driven decision-making, reflecting foundational machine learning concepts.
